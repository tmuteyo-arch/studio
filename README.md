# InnBucks Agent Onboarding Portal

This is a digital platform for onboarding financial agents, featuring a tiered workflow for Sales Leaders (ASL), Back Office Clerks, Supervisors, and Executive Management.

## Core Features

- **Multi-Role Dashboards**: Specialized views for ASL, Office Clerks, Supervisors, Compliance, and Management.
- **Dynamic Onboarding**: Support for 15+ account types including Individual, Sole Trader, Corporate, and Trust accounts.
- **Multi-Page Agreement Capture**: Physical document scanning and upload (up to 12 pages) with digital acknowledgement sign-off for the InnBucks Agreement.
- **Tiered Approval Workflow**: Integrated digital signatures for Supervisors and Executive Management.
- **Forensic Audit Dashboard**: Complete system traceability for Compliance users, including activity logs, rejection analysis, and application duration (TAT) tracking.
- **Document Repository**: Secure, high-detail PDF and image gallery for ASL users to review their submissions.
- **AI-Powered Verification**: Document extraction and cross-validation using Google Gemini.

## Getting Started

1. **Login**: Use the unified login screen to select your workspace.
2. **Onboard**: ASLs can start new applications or claim Customer Submissions from the portal.
3. **Audit**: Clerks and Supervisors verify documents and issue core registry IDs.
4. **Archive**: Accounts are finalized with 10-digit account numbers and stored in the global archive.
